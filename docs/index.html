flickspeed

The mouse speed overlay.

About

flickspeed is a program that captures mouse speed and translates it to recognizable, real-world speed values.

Features:

  • Support for mph, kph, and m/s
  • User-defined speed threshold and notification time
  • Small, non-intrusive and elegant overlay design
  • Process stays in system tray
